Function parser for C++: Speed tests

These tests were performed by running each specific test many times in a loop and then averaging the total time with the number of loops performed. The parsing, evaluation, evaluation after optimization, and optimization speeds were measured.

The test was run on a 2.4GHz Intel Core2 Quad using gcc 4.3.4 with the compiler options -O3 -ffast-math -march=native

(Note that the actual speeds may vary more than the amount of used significant digits in the list below from execution to execution. The most significant digit is nevertheless quite accurate.)

Function: "pow(x,14)+pow(y,8)"

  • Parse time: 0.69 us. (1 494 466 parses/s)
  • Eval time: 0.21 us. (4 828 993 evals/s)
  • Optimized: 0.049 us. (20 549 493 evals/s)
  • Optimization time: 40.1 us. (24 954 optimizes/s)
